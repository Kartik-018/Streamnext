1. **Clone the Repository:**
    ```bash

    ```

2. **Install Dependencies:**
    ```bash

    npm install
    ```

3. **Setup Environment Variables:**
    Create a `.env` file in the root directory and add your environment variables.

4. **Run the Application:**
    ```bash
    npm run dev
    ```
